Вот SQL запрос для поиска stationuuid с указанным URL, который отсутствует в banned_stations:

SELECT rs.stationuuid 
FROM radio_stations rs 
WHERE rs.url = 'https://radio.mp3islam.com/listen/abdulbasit/radio.mp3' 
  AND rs.stationuuid NOT IN (
    SELECT bs.stationuuid 
    FROM banned_stations bs 
    WHERE bs.stationuuid IS NOT NULL
  );
